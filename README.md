# Anime Quote Generator Website

This project is a simple web application that fetches and displays random anime quotes using the **Anime Chan API**. Built with **Node.js**, **Express**, **Axios**, and **EJS**, the application demonstrates how to integrate public APIs and present data in a user-friendly way. **Refresh the page to get a new random anime quote!**



## Features

- Fetch random anime quotes from the Anime Chan API  
- Display quotes dynamically on the web page  
- Beautiful and minimalistic UI design  
- Error handling for failed API requests  
- Presenting and manipulating data retrieved from APIs


## Technologies Used

- **Node.js** – JavaScript runtime  
- **Express.js** – Web framework for server-side logic  
- **Axios** – HTTP client for API requests  
- **EJS** – Templating engine for dynamic HTML  
- **HTML & CSS** – Frontend layout and styling  



## Installation

1. **Clone the repository**:

```bash
git clone <your-repo-url>
cd <project-directory>
```
2. **Install dependencies**:
   
```bash
npm install
```
3. **Start the server**:
   
```bash
node index.js
```

4. **Open in browser**:
   
```bash
http://localhost:3000
```
